# Проект автотестов с использованием Playwright и JavaFaker

Данный проект содержит пример автоматизированного теста, демонстрирующего перехват сетевых запросов и подмену ответов сервера с использованием сгенерированных данных.

## Запуск тестов

Проект написан на Java и не требует специального тестового раннера (используется `main` метод). Для запуска выполните следующие шаги:

1.  **Убедитесь, что установлены зависимости:**
    - Java 8 или выше.
    - Maven (для сборки проекта и управления зависимостями).

2.  **Соберите проект и скачайте зависимости:**
    В терминале, в корневой папке проекта выполните команду:
    ```bash
    mvn clean compile
